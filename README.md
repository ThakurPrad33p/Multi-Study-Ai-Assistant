# 🤖 Multi-Agent AI Study Assistant

A multi-agent AI-powered learning system that generates explanations and interactive quizzes in real time to help students understand concepts faster.

---

## 🚀 Features

- 📖 Simple, structured explanations for any topic  
- 📝 Interactive MCQ quizzes  
- ⚡ Instant feedback on answers  
- 🧠 Multi-agent architecture (Explanation + Quiz agents)  

---

## 🧩 How It Works

1. User inputs a topic or notes  
2. Explanation agent simplifies the content  
3. Quiz agent generates multiple-choice questions  
4. User interacts with quiz and gets instant feedback  

---

## ⚙️ Tech Stack

- Frontend: React  
- Backend: Node.js + Express  
- AI: Groq API (LLaMA-based models)
